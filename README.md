# Assignment 2
Console Runner Game 

This is a simple console-based game developed in C language.  
The player controls the character `O` and avoids falling obstacles `X`.

---

 Game Features
- Score increases after each dodge
- Player has 3 lives
- Difficulty (speed) increases as score increases
- Smooth movement without pressing Enter
- Option to play again after Game Over

---

 Controls
|Key| Action     |

| A | Move Left  |
| D | Move Right |
| Q | Quit Game  |

---
 How to Run
1. Open Terminal in the project folder  
2. Run:
```bash
clang game.c -o game
